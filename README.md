# 微语雀-语雀的第三方APP应用
一款基于Flutter跨平台的语雀客户端，目前已经完成了大部分功能，请耐心等待。

# 进度
最近语雀平台的数据类型和接口一直在改，不断地出现bug，因此我在考虑一种稳定的设计方案。

**重构**：由于刚开始对于flutter和dart的不熟悉，导致现在代码量日益趋大的同时维护相当困难，因此现在在对项目进行重构。
